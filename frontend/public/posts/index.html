My First Blog Post

Welcome to My First Blog Post

This is a sample blog post to demonstrate the comments functionality.

What is this blog about?

This blog is a proof of concept for a self-hosted blog with comments functionality. It uses:

  • Hugo for the static site generation
  • Express.js for the backend API
  • PostgreSQL for the database
  • JWT for authentication

How to use the comments system

  1. Register an account or log in
  2. Scroll to the bottom of any post
  3. Write your comment and submit
  4. Your comment will be posted immediately
